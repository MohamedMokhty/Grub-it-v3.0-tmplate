Infra Réseau Linux

Formation

Cet atelier de 03 jours est un apprentissage en conditions réelles conçu pour vous donner les compétences nécessaires pour gérer une infrastructure informatique professionnelle à l'aide d'outils open source à l'échelle d'une entreprise.

Cette formation essentiellement pratique est un véritable guide pour tous ceux qui veulent acquérir les aptitudes techniques pour déployer et administrer une infrastructure réseau et les services qui y sont associés.

Objectifs de la formation

À l’issue de la formation, le participant sera en mesure de :

  • Configurer des interfaces réseaux
  • Installer et Configurer un Serveur DHCP
  • Déployer un Serveur DNS
  • Configurer le Routage et la Passerelle (NAT)
  • Installer et Configurer un Partage Windows/Linux avec Samba
  • Déployer un Serveur Web (Apache, PHP) et des Hôtes Virtuels
  • Déployer un Serveur MariaDB et y créer des Bases de Données
  • Effectuer la Sauvegarde, la Réplication sur un site distant, et la Restauration de BD à l'aide de scripts
  • Etc.

Cet atelier vise aussi à édifier le participant sur quelques outils de sécurité d'une infrastructure réseau informatique (pare-feux, IDS, IPS, etc).

Plan du cours

Chapitre 1 : Introduction

  • 1.1 : Présentation du Système GNU/Linux
  • 1.2 : Etude de Quelques Commandes

Chapitre 2 : Infrastructure Réseau et Sécurité

  • 2.1 : Rappel : Adressage, Sous-Réseaux, Ports, Protocoles TCP/IP
  • 2.2 : Configuration des Interfaces Réseaux
  • 2.3 : DHCP : Installation et Configuration du Serveur
  • 2.4 : DHCP : Configuration Avancée
  • 2.5 : Déploiement du Serveur DNS
  • 2.6 : Routage et Passerelle (NAT)
  • 2.7 : Configuration de OpenSSH
  • 2.8 : Sécurité : Pare-feux, IDS, IPS

Chapitre 3 : Services Réseaux

  • 3.1 : Partage Sécurisé Windows/Linux avec Samba
  • 3.2 : Déploiement d'un Serveur FTP Sécurisé
  • 3.3 : Déploiement d'un Serveur Web (Apache, PHP) + Hôtes Virtuels
  • 3.4 : Mise en Place d'un Serveur NTP

Chapitre 4 : Déploiement de MariaDB

  • 4.1 : Installation et Configuration du Serveur
  • 4.2 : Exploitation Basique
  • 4.3 : Sauvegarde, Réplication et Restauration de Base de Données à l'aide de Scripts
